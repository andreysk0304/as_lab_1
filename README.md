## Формулировка задания

Реализовать бинарное дерево поиска (BST) с возможностью:
- Добавления узлов в дерево
- Удаления узлов из дерева
- Вывода дерева на экран
- Проверки симметричности структуры дерева (вариант 13)

Дерево считается симметричным по структуре, если левое и правое поддеревья являются зеркальным отражением друг друга по структуре (значения узлов могут отличаться, но структура должна быть симметричной).

## Описание выполненной работы

### Структура проекта

Проект состоит из следующих файлов:

#### 1. `tree.h` - Заголовочный файл

Определяет структуру `Node` и объявления функций для работы с бинарным деревом поиска.

#### 2. `tree.c` - Реализация функций

**Основные функции:**
- `createNode()` - создание узла
- `insert()` - рекурсивная вставка с соблюдением свойств BST
- `deleteNode()` - удаление узла (3 случая: без детей, с одним ребёнком, с двумя детьми)
- `printTree()` - рекурсивный вывод дерева с отступами
- `isSymmetricStruct()` / `isMirrorStruct()` - проверка симметричности структуры (вариант 13)
- `freeTree()` - освобождение памяти

```113:124:tree.c
bool isMirrorStruct(Node* node_left, Node* node_right) {
    if (node_left == NULL && node_right == NULL) return true;
    if (node_left == NULL || node_right == NULL) return false;

    return isMirrorStruct(node_left->left, node_right->right) &&
           isMirrorStruct(node_left->right, node_right->left);
}

bool isSymmetricStruct(Node* root) {
    if (root == NULL) return true;
    return isMirrorStruct(root->left, root->right);
}
```

#### 3. `main.c` - Основная программа

Интерактивное меню с операциями: добавление, удаление, вывод дерева, проверка симметричности.

#### 4. `tests.c` - Тесты

Тесты для проверки создания узлов, вставки, удаления и проверки симметричности.

## Как запустить

**Запуск основной программы:**
```bash
make main
```

**Запуск тестов:**
```bash
make test_all
```

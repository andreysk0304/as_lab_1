## Формулировка задания

Составить программу для сортировки таблицы заданным методом (простая вставка) и двоичного поиска по ключу. Программа должна:
- Вводить элементы неупорядоченной таблицы (в т.ч. из файла)
- **Проверять работу сортировки в трёх случаях:** (1) элементы изначально упорядочены; (2) элементы в обратном порядке; (3) элементы не упорядочены (допускается генерация псевдослучайных чисел). Для каждого вызова процедуры сортировки — печать исходного состояния таблицы и результата сортировки
- После сортировки вводить ключи и для каждого выполнять двоичный поиск, печатая найденные элементы
- Использовать тестовые данные из текстовых файлов, при необходимости — переадресацию ввода-вывода

## Описание выполненной работы

### Структура проекта

#### 1. `table.h` — заголовочный файл

Определяет структуру `Table` (массивы ключей и значений, размер) и объявления функций для работы с таблицей.

#### 2. `table.c` — реализация

**Основные функции:**
- `initTable()` — инициализация пустой таблицы
- `addRow()` — добавление строки (ключ, значение)
- `freeTable()` — освобождение памяти
- `printTable()` — вывод таблицы
- `readTableFromFile()` — чтение из файла (формат: ключ пробел значение)
- `swapRows()` — обмен двух строк
- `copyTable()` — копирование таблицы
- `insertionSort()` — сортировка вставками по ключам
- `binarySearch()` — бинарный поиск по ключу
- `reverseTable()`, `shuffleTable()` — разворот и перемешивание

#### 3. `main.c` — основная программа

Интерактивное меню: чтение из файла, добавление/удаление записей, вывод таблицы, запись в файл, бинарный поиск по ключу, **п.7 — проверка сортировки в трёх случаях** (упорядоченная таблица, обратный порядок, неупорядоченная таблица; для каждого печатаются исходная таблица и таблица после сортировки).

#### 4. `tests.c` — тесты

Тесты для `initTable`, `addRow`, `swapRows`, `copyTable`, `insertionSort`, `binarySearch`, `reverseTable`, `shuffleTable`, `freeTable`.

## Как запустить

**Запуск основной программы:**
```bash
make main
```

**Запуск тестов:**
```bash
make test_all
```

## Пример работы

Программа читает файл `input.txt`. Пример содержимого:

```
1 hello
2
3 hi
4 это фокус
5
```

**Вывод программы:**
```
Таблица

1  hello
2  
3  hi
4  это фокус
5  
```

**Проверка сортировки в 3 случаях (п.7 меню):** программа выводит для каждого случая исходную таблицу и таблицу после сортировки: (1) уже упорядоченная, (2) обратный порядок, (3) неупорядоченная (перемешивание через rand). После сортировки можно вводить ключи и выполнять поиск (п.6).

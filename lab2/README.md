## Формулировка задания

Реализовать таблицу «ключ–значение» с возможностью:
- Чтения данных из файла (пары ключ и значение через пробел в строке)
- Сортировки таблицы по ключам (сортировка вставками)
- Поиска по ключу (бинарный поиск)
- Вывода таблицы на экран

Дополнительно реализованы: копирование таблицы, обмен строк, разворот таблицы, перемешивание (shuffle).

## Описание выполненной работы

### Структура проекта

#### 1. `table.h` — заголовочный файл

Определяет структуру `Table` (массивы ключей и значений, размер) и объявления функций для работы с таблицей.

#### 2. `table.c` — реализация

**Основные функции:**
- `initTable()` — инициализация пустой таблицы
- `addRow()` — добавление строки (ключ, значение)
- `freeTable()` — освобождение памяти
- `printTable()` — вывод таблицы
- `readTableFromFile()` — чтение из файла (формат: ключ пробел значение)
- `swapRows()` — обмен двух строк
- `copyTable()` — копирование таблицы
- `insertionSort()` — сортировка вставками по ключам
- `binarySearch()` — бинарный поиск по ключу
- `reverseTable()`, `shuffleTable()` — разворот и перемешивание

#### 3. `main.c` — основная программа

Инициализирует таблицу, читает данные из `input.txt`, выводит таблицу и освобождает память.

#### 4. `tests.c` — тесты

Тесты для `initTable`, `addRow`, `swapRows`, `copyTable`, `insertionSort`, `binarySearch`, `reverseTable`, `shuffleTable`, `freeTable`.

## Как запустить

**Запуск основной программы:**
```bash
make main
```

**Запуск тестов:**
```bash
make test_all
```

## Пример работы

Программа читает файл `input.txt`. Пример содержимого:

```
1 hello
2
3 hi
4 это фокус
5
```

**Вывод программы:**
```
Таблица

1  1  hello
2  2  
3  3  hi
4  4  это фокус
5  5  
```

**Пример с сортировкой и поиском (логика в коде):** после `insertionSort` таблица упорядочена по ключам; `binarySearch(&table, "3")` возвращает индекс строки с ключом `"3"`.

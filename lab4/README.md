## Формулировка задания

Построить упорядоченное дерево для хранения и поиска элементов с уникальными строковыми ключами и вещественными значениями. Для лабораторной работы реализовано B-дерево порядка 2.

Поддерживаются операции:
- добавление узла по ключу;
- удаление узла по ключу;
- печать дерева;
- поиск значения по ключу;
- выполнение команд из текстового файла;
- запись результатов выполнения команд в текстовый файл.

Ключ задаётся непустой последовательностью латинских букв длиной не более 6 символов.

## Структура проекта

1. `tree.h`, `tree.c` — структура B-дерева, вставка, удаление, поиск, печать и обработка команд из файла.
2. `main.c` — консольное меню для ручной работы с деревом и запуска команд из файла.
3. `tests.c` — тесты на вставку, поиск, удаление, работу с набором из 20+ ключей и файловый сценарий.
4. `input.txt` — пример входного файла с командами.
5. `Makefile` — сборка и запуск основной программы и тестов.

## Как запустить

Запуск основной программы:

```bash
make main
```

Запуск тестов:

```bash
make test_all
```

## Формат команд из файла

Команда записывается в формате:

```text
<номер операции> [<ключ>] [<значение>]
```

Где:
- `1 key value` — добавить узел;
- `2 key` — удалить узел;
- `3` — вывести дерево;
- `4 key` — найти значение по ключу.

Для каждой команды в выходной файл записываются:
- исходная команда;
- результат её выполнения.

## Пример входного файла

```text
1 mango 10
1 apple 1
1 peach 8
1 berry 2
3
4 apple
2 berry
3
```

## Что проверяют тесты

- вставку ключей и поиск значений;
- удаление листьев и внутренних ключей;
- работу дерева на наборе больше чем из 20 узлов;
- обработку команд из файла и запись результата в выходной файл;
- проверку ограничений на ключи.
